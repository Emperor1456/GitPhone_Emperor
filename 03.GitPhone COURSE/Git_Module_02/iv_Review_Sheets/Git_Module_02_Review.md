# 🔁 Module 02 – Final Review (Branching & Collaboration)

## 🧪 Consolidation Challenge
Create a new folder `review02`, initialize a repo, and complete these tasks.

1. Create a file `main.txt` with "line1", commit it.
2. Create a branch `feature`. Switch to it.
3. In `feature`, modify `main.txt` to add "line2". Commit.
4. Switch back to `main`. Merge `feature` (fast‑forward expected).
5. Create another branch `feature2`. Switch to it, change `main.txt` to "line3". Commit.
6. Switch to `main`, also change `main.txt` to "line4". Commit.
7. Merge `feature2` into `main`. Resolve the conflict, keeping both lines.
8. Experiment with `git stash`: make an uncommitted change, stash it, then pop it.
9. Use `git restore` to discard a change.
10. Practice `git reset --soft HEAD~1` and then recommit.

After the challenge, clean up by deleting `review02`.
