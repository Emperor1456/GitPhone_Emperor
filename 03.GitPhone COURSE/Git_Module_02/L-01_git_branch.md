📝 Lesson 1 – git branch

📖 What it does:
→ git branch creates a new line of development.
→ Branches let you work on features without touching main.
→ The original branch stays safe while you experiment.

🛠️ Practice:
In git-practice, create a branch and list all branches.

🚀 Commands:
$ git branch feature-x
→ Create a branch named feature-x.
$ git branch
→ List all branches (current branch has a *).
$ git branch -d feature-x
→ Delete the branch (clean up after yourself).
-------------------------------------

Use branches for every new feature or fix.
