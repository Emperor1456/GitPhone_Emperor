📝 Lesson 2 – git switch

📖 What it does:
→ git switch moves you between branches.
→ It updates your working files to match the branch.
→ Always switch before starting work.

🛠️ Practice:
Create a branch and switch into it.

🚀 Commands:
$ git branch dev
→ Create a branch called dev.
$ git switch dev
→ Move to the dev branch.
$ git branch
→ Confirm you are on dev (* appears).
$ git switch main
→ Return to main.
$ git branch -d dev
→ Delete dev when you're done.
-------------------------------------

`git checkout` does the same thing,
but `git switch` is clearer.
