📝 Lesson 8 – First push to GitHub

📖 What it does:
→ Connect your local repo to GitHub and upload it.
→ git remote add origin <url> – links the two.
→ git push -u origin main – sends your commits up.

🛠️ Practice:
First, create a repo on GitHub (no README, no .gitignore).
Copy its URL. Then in git-practice:

🚀 Commands:
$ git remote add origin <your-url>
→ Link local to remote.
$ git branch -M main
→ Rename branch to main.
$ git push -u origin main
→ Upload and set upstream.
-------------------------------------

Your code is now on GitHub. Refresh the page to see it.
Every later push is just git push.
