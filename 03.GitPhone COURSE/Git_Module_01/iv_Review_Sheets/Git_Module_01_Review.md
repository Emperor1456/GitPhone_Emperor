# 🔁 Module 01 – Final Review (Git Foundations)

## 🧪 Consolidation Challenge
Perform every step in Termux without looking up commands. Use a new folder `review01`.

1. Create a new directory `review01` and navigate into it.
2. Initialize a Git repository.
3. Create a file `readme.txt` with content "Hello Git".
4. Stage and commit with message "Initial commit".
5. Check the commit log to see your commit.
6. Check the repository status (should be clean).
7. Create a `.gitignore` that ignores all `.log` files.
8. Create a dummy `error.log` file and confirm it is ignored by `git status`.
9. Make a change to `readme.txt`, view the diff with `git diff`, then stage and commit.
10. (Optional) Add a remote named `origin` pointing to a test repo and push.

After completion, delete `review01`.
