📝 Lesson 4 – git log

📖 What it does:
→ git log shows the history of commits.
→ It’s the timeline of your project.
→ Use it to see where you’ve been.

🛠️ Practice:
Run these in git-practice:

🚀 Commands:
$ git log
→ Show detailed commit history (press q to exit).
$ git log --oneline
→ Show compact one‑line history.
$ git log --oneline -2
→ Show only the last two commits.
-------------------------------------

The first command shows full details, the second a compact list,
the third limits entries. Press q to exit the log view.
