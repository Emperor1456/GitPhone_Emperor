📝 Lesson 4 – git log

What it does:
git log shows the history of commits – who made them, when, and what changed.
It’s the timeline of your project. Use it to see where you’ve been.

Practice
Run these in git-practice:

🚀 Commands:
· git log
· git log --oneline
· git log --oneline -2
---------------------

- The first command shows detailed info.
- The second gives a compact list.
- The third limits to the last two entries.
- Press q to exit the full log view.
