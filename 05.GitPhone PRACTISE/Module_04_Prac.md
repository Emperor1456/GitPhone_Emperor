📝 Capstone – Real Open‑Source Contribution

🛠️ Task:
Find a real project, fork it, make a meaningful fix,
and submit a pull request entirely from your phone.

🚀 Steps:
→ Browse GitHub and find a project with "good first issue" labels.
→ Fork it.
→ Clone your fork.
→ Create a branch.
→ Make the fix.
→ Commit with a clear message.
→ Push to your fork.
→ Open a pull request.
→ Respond to any review feedback.
→ Get it merged.
