# 👑 GitPhone Emperor – Master Git from Your Phone
**A free, mobile‑first Git & GitHub curriculum. No laptop needed.**

Built entirely on an Android phone using Termux and Acode.  
This course takes you from absolute zero to a professional Git user – one typed command at a time.  
Now open‑source for everyone.

---

## 📂 What’s inside

Think of this repo as a complete Git learning kit. Every folder has a clear job:

**01.GitPhone PLUGIN/** – Your starting point.  
Setup guides that turn an Android phone into a full Git machine (Termux, Acode, Git, GitHub).

**02.GitPhone SYLLABUS/** – The big picture.  
The complete course outline plus the AI prompt that becomes your personal Git mentor.

**03.GitPhone COURSE/** – The core lessons.  
Every topic is a separate `.md` file, numbered in order. Start at `L-01` and work forward.

**04.GitPhone NOTES/** – Quick revision.  
One‑page cheat sheets for each module, designed to fit your phone screen perfectly.

**05.GitPhone PRACTISE/** – Prove your skills.  
Hands‑on practice sets that mix everything from the module into real‑world Git challenges.

---

**What you’ll learn, in order:**  
Git Basics → Commits & History → GitHub & Remotes → Branching & Merging → Collaboration via Pull Requests → Resolving Conflicts → Rebasing & Cherry‑picking → Open‑Source Contribution → Advanced Debugging with Git.

For the full topic‑by‑topic checklist, open `02.GitPhone SYLLABUS/GitPhone_Curriculum.md`.

   ---

## 🚀 How to start

1. Read the `SETUP_GUIDE.md` inside `01.GitPhone PLUGIN/`. It installs Termux, Acode, Git, and links your phone to GitHub.
2. Copy the AI prompt from `02.GitPhone SYLLABUS/GitPhone_AI_Prompt.md` into your AI assistant (DeepSeek, ChatGPT, etc.). It becomes your personal Git mentor.
3. Open `L-01` inside `03.GitPhone COURSE/GitPhone_Module_01/` and start typing.

No copy‑paste – you type every command yourself. That’s how real mastery happens.

---

## 🧠 The method

- **Two bricks at a time** – exactly one or two small concepts per lesson, never overwhelming.
- **Phone‑first design** – all content fits a phone screen, no sideways scrolling.
- **AI‑assisted, not AI‑dependent** – you understand every single command you run.
